筆者はドイツの行政問題やスタジオ移転のため、約10年ぶりにSuperboothへの参加を断念したが、ロンドンで開催された「London Synth & Pedal Expo 2026」に参加し、その様子をレポートしている。このイベントは小規模ながらも活気があり、多数のメーカーが参加し、特にMonk ProjectのTMA1やGenki InstrumentsのKatlaといった革新的な製品が注目を集めた。最新のシンセサイザーとペダルのトレンドがわかる。

London Synth & Pedal Expoは、Hackney Wickの会場で開催され、ペダルとシンセサイザーの2つの異なる部屋で構成されていた。Arturia、Moog、Korgなどの大手から、Melbourne Instruments、1010 Music、Dreadboxといった個性的なメーカーまで、約50社が出展。特に、20種類以上のシーケンサーモデルを持つジェネレーティブシーケンサー「Monk Project TMA1」や、アイスランドのGenki Instrumentsによる5000ドル超のポリフォニックラックマウントシンセ「Katla」が強い印象を残した。Katlaは火山溶岩をノブやサイドプレートに使うなどユニークな特徴を持ち、その革新性と有機的なサウンドが高く評価されている。また、スマートフォンを3Dモーションコントローラーにする無料アプリ「Flowfal」も紹介されている。

Marian (BR)：「ハウスミュージックのルーツを学べ」

DJ兼プロデューサーのMarian (BR)が、自身の音楽キャリアとハウスミュージックへの深いルーツについて語っている。シカゴハウスに強く影響を受け、シンプルながらパワフルなグルーヴを追求する彼のサウンドは、Jamie JonesやThe Martinez BrothersといったトップDJたちにも支持され、イビサでの重要なギグへと繋がった。成功への道のりと、若手クリエイターへのアドバイスを提供している。

Marian (BR)は、初期のハウスミュージック、特にシカゴハウスのサンプルチョップ、ボーカル、ピアノ、オルガン、サチュレートされたドラムサウンドからインスピレーションを得ているという。彼のトラックは、過度な複雑さを避け、ダンスフロアでのインパクトを最優先に制作されており、「Swing Bom」がPacha IbizaのMusic Onシーズンでフィーチャーされたことはキャリアの大きな転機となった。また、彼はHot CreationsやElrow Musicといった著名レーベルからのリリース経験を持ち、南米やヨーロッパでのツアーを通じて、サウンドをダンスフロアに最適化する重要性を学んだ。キャリアパスを追求する人々へのアドバイスとして、「誰も聞いていなくても音楽を作るか？」という問いかけと、ルーツを学び、クラブに足を運び、拒絶に備えることの重要性を強調している。

DTM初心者のためのオーディオインターフェイスの選び方2026

DTMを始める上で不可欠な機材であるオーディオインターフェイスについて、初心者向けに選び方を解説する記事。多種多様な製品が存在する中で、何を基準に選べば良いのか、その機能や重要性を分かりやすく説明している。DTM環境構築の第一歩を踏み出す上で非常に役立つ情報を提供する。

オーディオインターフェイスは、コンピュータと楽器・マイクを繋ぎ、高音質で音を録音・再生するためのデバイス。記事では、主に「入出力端子の種類と数」「音質」「接続方式（USB、Thunderboltなど）」「付属ソフトウェア」「価格帯」といった選択基準を詳細に解説すると考えられる。特にDTM初心者にとっては、自身の制作スタイルや予算に合った製品を見つけることが重要であり、この記事は最初の機材選びにおける羅針盤となるだろう。
