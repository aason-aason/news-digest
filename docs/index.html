Cotorro AudioがWindows用無料マルチバンドハーモニックエキサイター「P1E-RU」をリリース

Cotorro Audioが、Windows向けの無料マルチバンドハーモニックエキサイタープラグイン「P1E-RU」をリリースした。このプラグインは、音源信号を3つの固定周波数帯域（低域、中域、高域）に分割し、それぞれに独立してサチュレーションを適用できる。これにより、サウンドに倍音を加え、クリアさやパンチを増強することが可能となり、Windowsユーザーは無償で高品質なサウンドメイクツールを利用できる。

Going Off Grid: スウィングとは何か、そしてどう追加するか

DTMにおいて楽曲にグルーヴと生命力を与える「スウィング」の概念とその具体的な適用方法を解説するチュートリアル。このガイドでは、スウィングの理論的背景からAbleton LiveのGroove Poolを使った実践的な方法まで、ステップバイステップで説明し、ダフト・パンクやフレッド・アゲインなどの楽曲を例にその効果を実感させる。スウィングを理解し使いこなすことで、より躍動感のあるリズムを制作できるようになるだろう。

スウィングとは、均等に配置されたリズムの弱い拍をグリッドからずらすことで、独特の「ノリ」を生み出すテクニックである。記事では、まず基本的なスウィングの定義を、16分音符のハイハットを例に、50%（ストレート）から52%、64%、73%とずらしていく音源と図解で分かりやすく解説する。Ableton LiveのGroove Pool機能を使用することで、手動でノートをずらす手間なく、クリップ全体にスウィングを適用する方法が示される。また、Groove PoolのTiming、Base、Quantise、Random、Velocityといった他の設定についても言及され、スウィングがドラムだけでなく、リード、ベースライン、サンプルなどトラックのあらゆる要素に適用可能であることが説明されている。
