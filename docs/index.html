Cotorro AudioがWindows向け無料マルチバンドハーモニックエキサイター「P1E-RU」をリリース

Cotorro Audioが、Windows向けの無料マルチバンドハーモニックエキサイタープラグイン「P1E-RU」をリリースした。このツールは、オーディオ信号を3つの固定周波数帯域に分割し、それぞれに独立してサチュレーションを適用できるため、サウンドに輝きと厚みを加えるのに役立つため、今日中に確認すべき重要な情報です。

P1E-RUは、20Hz～350Hzの低域、350Hz～2.5kHzの中域、2.5kHz～22kHzの高域に信号を分け、各帯域で個別にハーモニック（倍音）を追加できる。これにより、ボーカルや楽器トラック、ミックス全体にパンチや存在感を与えることが可能になり、無料でありながらプロフェッショナルなサウンドメイクに貢献する。
