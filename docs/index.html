FathomがGranolaに対抗し、ボットなしの会議モードを追加

AI会議アシスタントのFathomが、ボットを使わずに会議を記録できる「ボットなしモード」を導入した。これにより、ユーザーはビデオ、オーディオのみ、文字起こしのみのモードを選択でき、競合サービスGranolaとの差別化を図る。この機能は、AIボットが会議に参加することに抵抗があるユーザーや、特定の規制下での利用を可能にし、より幅広い利用シーンに対応する。

FathomはこれまでAIボットを会議に参加させて記録していたが、今回のアップデートでボットを会議に招待せず、自動でビデオ、オーディオ、文字起こしを記録できるようになった。このモードは特に、セキュリティやプライバシーに敏感な企業、またはAIボットの存在を避けたいユーザー層をターゲットにしている。競合のGranolaは既に同様の機能を提供しており、Fathomはこれに追随しつつ、ユーザーの選択肢を広げることで市場での競争力を高めようとしている。
