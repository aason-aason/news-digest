デンソーがロームの買収提案を撤回しました。トヨタの懸念や半導体3社（ルネサス、ソニー、キオクシア）陣営の警戒といった逆風が原因と考えられます。これにより、デンソーは成長の核となる半導体戦略の練り直しを迫られています。

自動車部品大手のデンソーは、半導体メーカーのロームへの買収提案を撤回しました。この撤回には、親会社であるトヨタ自動車からの懸念や、国内の主要半導体メーカー3社（ルネサスエレクトロニクス、ソニーグループ、キオクシアホールディングス）の警戒感といった、複数の要因が影響したと見られています。EV（電気自動車）や自動運転技術の進化に伴い、半導体の重要性が増す中で、デンソーは半導体事業の成長戦略の見直しを余儀なくされており、今後の新たな戦略が注目されています。
