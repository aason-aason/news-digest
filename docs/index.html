ティム・クック氏、Apple CEOを退任、ジョン・ターナス氏が後任に

ティム・クック氏がAppleのCEOを退任し、ジョン・ターナス氏が後任に就任しました。この交代は予想されていたもので、クック氏の長期にわたる在任期間とAppleに与えた多大な影響に幕を下ろすものです。クック氏は、創業者から受け継いだ会社を4兆ドル規模のビジネスへと成長させました。
