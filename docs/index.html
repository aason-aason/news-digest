Echoe、ブラウザベースのロスレスオーディオファイル共有プラットフォームをローンチ

ベルリン拠点のエンジニア、パトリック・スクーグ氏が開発したEchoeは、圧縮なしでファイル配信、フィードバック、バージョン管理を効率化するブラウザベースのロスレスオーディオファイル共有プラットフォームをリリースしました。これにより、ミュージシャンやプロデューサーは、複数のサービスに分散していた作業を統合できます。

Echoeは、WAVやAIFFといったロスレスオーディオファイルをブラウザ上で直接ストリーミングし、圧縮なしで共有できるサービスです。タイムスタンプ付きのフィードバック機能も備えており、曲の特定の部分に対するコメントを直接残すことができます。これは、従来のDropboxやGoogle Driveなどの汎用ファイル共有サービスが、オーディオ制作ワークフローのニーズを十分に満たしていないという課題に対応するものです。無料プランも提供されており、より高度な機能は有料プランで利用可能です。
